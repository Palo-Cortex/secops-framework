### Version 3.3.2

This release introduces the first formal release notes for the **SOC Optimization Unified** pack and represents continued maturation of the SOC Framework content library. Version **3.3.2** focuses on improving platform compatibility, stabilizing core automation components, and preparing the pack for broader distribution and contribution workflows.
